MapApp, kullanıcıların harita üzerinde konum bazlı işlemler yapmasını sağlayan bir mobil uygulamadır. Kullanıcı dostu arayüzü ile konum görüntüleme, işaretleme ve etkileşimli harita deneyimi sunar.

## 🚀 Features

- 📍 Harita üzerinde konum görüntüleme
- 📌 Marker (işaretçi) ekleme
- 🧭 Kullanıcı konumunu alma
- 🔍 Harita üzerinde gezinme ve zoom
- 📱 Modern ve kullanıcı dostu arayüz

## 🛠️ Technologies Used

- Flutter
- Google Maps API
- Dart
